About ANODE-EVAL

ANODE-EVAL is a framework for evaluating AI coding agents. It runs agents on programming tasks and measures their performance using automated tests.

Supported Agents

  • - Claude Code (Opus, Sonnet, Haiku)
  • - Codex (GPT-5.2, O3)
  • - OpenCode (Qwen Coder)

Test Harnesses

  • - Cargo (Rust)
  • - npm (JavaScript/TypeScript)
  • - pytest (Python)
  • - Go test

Features

  • - Kubernetes orchestration
  • - Parallel execution
  • - Automated scoring
  • - Result comparison
